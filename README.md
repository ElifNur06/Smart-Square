# Akıllı Kare (Smart Square) 🧠✨

Akıllı Kare, dijital varlıklarınızı yönetmenizi, karmaşık içerikleri yapay zeka ile analiz etmenizi ve tüm bu süreçleri akıllı bir takvim ile planlamanızı sağlayan, hibrit bir AI asistan uygulamasıdır.

## 🚀 Temel Özellikler

- **Dinamik Mod Bazlı Analiz:** Genel, Etkinlik, Matematik ve Yazılım modları ile kategorisel içerik yönetimi.
- **Yapay Zeka Analisti:** Google Gemini API entegrasyonu ile metin, görsel ve matematiksel problem analizi.
- **Akıllı Takvim:** `add_2_calendar` entegrasyonu ile algılanan etkinlikleri doğrudan takvime işleme.
- **Sesli Erişim (TTS):** Üretilen analizleri `flutter_tts` ile sesli olarak dinleme imkanı.
- **Premium Erişim:** Özel analiz modları için katmanlı erişim ve abonelik kontrolü.
- **Gelişmiş Arama:** Geçmiş analizler içerisinde metin tabanlı hızlı filtreleme.
- **Sosyal Entegrasyon:** `share_plus` ve `url_launcher` ile sonuçları paylaşma ve kaynaklara web üzerinden erişim.

## 🛠 Teknik Altyapı (Tech Stack)

| Kategori | Teknoloji / Kütüphane |
| :--- | :--- |
| **Framework** | Flutter (Dart) |
| **Veritabanı** | SQFlite (SQLite) |
| **Yapay Zeka** | Google Gemini API (gemini-2.5-flash) |
| **Takvim** | `add_2_calendar` |
